# Techstone
Desenvolvimento web

## Instalação

### Digite a linha de comando na raíz do projeto:

composer update


### Digite as linhas de comando na pasta public localizada na raíz do projeto:

npm install jquery

npm install bootstrap

npm install popper.js

npm install admin-lte --save
